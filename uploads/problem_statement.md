# Problem Statement: CodeWhisper

## Domain
**AI Powered Developer Tools and Productivity Systems**

## Problem Statement

Developers and students preparing for technical interviews, coding assessments, and competitive programming contests often rely on online solutions when they get stuck on Data Structures and Algorithms (DSA) problems. While these solutions help users arrive at the correct answer quickly, they frequently encourage answer copying rather than genuine problem-solving, limiting long-term learning and skill development.

To address this challenge, build **CodeWhisper**, an AI-powered hint engine that acts as a learning companion rather than a solution generator. The platform should allow users to paste any coding or DSA problem and receive a sequence of intelligent, progressive hints that guide them toward the solution without revealing the final answer. The system should analyze the problem, identify underlying concepts and patterns, and provide Socratic-style guidance that encourages critical thinking and independent problem solving.

Built using **Python**, **Flask**, and modern **LLM APIs such as OpenAI or Claude**, CodeWhisper should adapt the depth and complexity of hints based on user interactions and responses. The platform should also maintain a history of solved problems, track learning progress, and recommend similar practice problems to strengthen concept mastery. The goal is to create an AI-powered productivity tool that transforms how programmers learn, practice, and improve their problem-solving skills while maintaining academic integrity and fostering deeper understanding.
